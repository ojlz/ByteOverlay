# ByteOverlay

ByteOverlay é uma overlay de estatísticas desenvolvida para o servidor Mush, compatível com Minecraft Forge 1.8.9.

O projeto foi criado com foco em fornecer informações relevantes durante as partidas, permitindo uma análise rápida dos jogadores presentes na sessão através de uma interface altamente configurável e integrada ao jogo.

O desenvolvimento contou com o auxílio de Inteligência Artificial, utilizando principalmente o Gemini CLI como ferramenta de apoio durante a implementação, testes e refinamento de recursos.

## Recursos

[+] Overlay de estatísticas em tempo real para jogadores do servidor Mush.

[+] Detector automático de jogadores com nick falso identificados como `[NICKED]`.

[+] Detector automático de membros da equipe administrativa e moderação.

[+] Sistema de identificação automática de possíveis snipers com base em suas estatísticas.

[+] Cadastro manual de snipers através do arquivo de configurações.

[+] Alertas visuais no chat quando um nick falso, staff ou sniper é encontrado.

[+] Alertas sonoros configuráveis para eventos importantes.

[+] Nametag acima da cabeça dos jogadores exibindo estatísticas e informações relevantes.

[+] Exibição de informações de clã mesmo quando a opção ClanTagDisplay estiver desativada.

[+] Transparência do plano de fundo totalmente configurável.

[+] Sistema de cores personalizável para todos os elementos da interface.

[+] Teclas de atalho configuráveis diretamente através das configurações do mod.

[+] Interface projetada para rápida leitura durante combates e partidas competitivas.

[+] Configuração simplificada através de arquivos locais.

## Compatibilidade

* Minecraft 1.8.9
* Minecraft Forge 1.8.9
* Servidor Mush

## Configuração

Todas as opções do ByteOverlay podem ser ajustadas através dos arquivos de configuração do mod, incluindo:

* Transparência da interface.
* Cores dos elementos.
* Teclas de atalho.
* Alertas sonoros.
* Lista personalizada de snipers.
* Exibição de estatísticas.
* Configurações das nametags.

## Aviso

Este projeto não possui qualquer vínculo oficial com o servidor Mush, sua equipe administrativa ou seus desenvolvedores.

ByteOverlay é um projeto independente criado pela comunidade com fins de personalização e aprimoramento da experiência de jogo.
